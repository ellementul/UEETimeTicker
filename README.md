# UEE Time Ticker

## Description
This module is part of the counting time system in UEE.

#### The module run by start event:
```js
  const { 
    events: { start } 
  } = require("@ellementul/uee-timeticker") 
```

#### The module generate time event every tick:
```js
  const { 
    events: { time } 
  } = require("@ellementul/uee-timeticker")
```
